## Installation

###  **Install Manually:**
- Download the **IconicBlocks** plugin `.zip` file.
- In your WordPress dashboard, go to **Plugins** > **Add New**.
- Click on **Upload Plugin**, select the `.zip` file, and hit **Install Now**.
- Activate the plugin after installation.
